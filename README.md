# 🌸 鸢尾花品种预测项目

这是一个基于 **Scikit-learn** 和 **逻辑回归模型** 的鸢尾花品种分类项目。项目使用经典的鸢尾花数据集，通过数据探索、可视化、模型训练与评估，最终构建一个能够准确预测鸢尾花品种的分类器。

---

## 📁 项目结构

```
鸢尾花品种预测.ipynb          # 主 Notebook 文件
iris_classifier.pkl           # 训练好的模型文件
iris_metadata.pkl             # 特征和目标名称元数据
README.md                     # 项目说明文档
```

---

## 🎯 项目目标

- 加载并探索鸢尾花数据集
- 可视化数据分布和特征关系
- 使用逻辑回归模型进行分类
- 评估模型性能并保存模型
- 提供新数据的预测功能

---

## 📊 数据集

使用 Scikit-learn 内置的 **鸢尾花数据集**，包含：

- **150 个样本**，3 个品种各 50 个
- **4 个特征**：
萼片长度
  - 花萼宽度（sepal width） 
  - 花瓣长度（petal length）
  - 花瓣宽度（petal width）
- **3 个目标品种**：
  - Setosa（山鸢尾）
  - Versicolor（变色鸢尾）
  - Virginica（维吉尼亚鸢尾）

## 🔧 自定义预测
